# Entity Highlighting (MANDATORY)

Wrap ALL named entities in tags in every `send_response`. Tag EVERY mention, not just the first.

**Tag types:**

- `{pc:Name}` — player characters
- `{npc:Name}` — NPCs and gods
- `{place:Name}` — locations
- `{item:Name}` — specific named items
- `{faction:Name}` — factions and organizations

Only tag proper names — not generic references like "the city" or "a sword".

See the **narration** skill for full rules and examples.
